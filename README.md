# Smart-Campus-Resource-Utilization-Tracker

A smart campus management system to monitor and optimize the use of labs, classrooms, and library resources in real-time.

---

## 1️⃣ Problem Statement

Most colleges face:

* Overcrowded or underutilized labs, classrooms, and equipment
* Manual tracking of usage
* Students wasting time finding free spaces

**Impact:** Poor scheduling, resource wastage, no data-driven insights.

---

## 2️⃣ Solution

A **centralized platform** that:

* Tracks resource usage live and historically
* Provides analytics dashboards
* Predicts peak and idle hours
* Helps students, faculty, and admins make data-driven decisions

**Goal:** Optimize utilization, reduce crowding, and improve planning.

---

## 3️⃣ Users

| User     | Benefits                                 |
| -------- | --------------------------------------- |
| Students | View free labs, library seats, equipment |
| Faculty  | Book resources efficiently               |
| Admin    | Access analytics for planning            |

---

## 4️⃣ Key Features

* **Real-Time Status:** Free, occupied, reserved; live occupancy %
* **Resource Booking:** Slot-based reservations, conflict detection
* **Analytics Dashboard:** Daily/weekly/monthly usage, peak vs idle hours
* **Predictive Forecasting:** Suggests optimal usage times
* **Admin Insights:** Recommendations to merge/add resources

---

## 5️⃣ Tech Stack

**Frontend:** React.js, Tailwind CSS, Recharts  
**Backend:** Node.js + Express (REST APIs), JWT Auth  
**Database:** PostgreSQL (or MongoDB)  
**ML & Analytics:** Python, Pandas, Scikit-learn, Matplotlib  
**Deployment:** AWS / Railway / Render, Docker

---

## 6️⃣ How It Works

* Users check in via **QR codes** or login manually
* System tracks resource occupancy
* Analytics and ML engine predict usage trends
* Admin and users access dashboards for insights
