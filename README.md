# Smart-Campus-Resource-Utilization-Tracker

A smart campus management system to monitor and optimize labs, classrooms, and other resources in real-time.

---

## 1️⃣ Problem Statement

Most colleges face:

* Overcrowded or underutilized labs, classrooms, and equipment
* Manual tracking of usage
* Students struggling to find free spaces

**Impact:** Poor scheduling, wasted resources, and lack of insights.

---

## 2️⃣ Solution

A **centralized platform** that:

* Tracks resource usage in real-time
* Sends notifications for updates and bookings
* Allows teachers to book resources and merge sections
* Lets admins manage the entire system

**Goal:** Optimize utilization, reduce conflicts, and improve campus management.

---

## 3️⃣ User Roles & Access

| Role     | Features                                         |
| -------- | ----------------------------------------------- |
| Student  | View labs & rooms, receive notifications       |
| Teacher  | Book resources, merge sections                  |
| Admin    | Full control: manage users, resources, analytics |

---

## 4️⃣ Key Features

* **Real-Time Resource Status:** See available labs and rooms instantly
* **Notifications:** Students get alerts when resources change
* **Resource Booking (Teacher):** Schedule labs, classrooms, and merge sections
* **Admin Control:** Add/edit/remove resources and manage all users
* **Analytics Dashboard:** Track usage, detect underutilized spaces

---

## 5️⃣ Tech Stack

**Frontend:** React.js, Tailwind CSS, Recharts  
**Backend:** Node.js + Express, REST APIs  
**Database:** MySQL  
**Deployment:** AWS / Railway / Render  

---

## 6️⃣ How It Works

* Students log in to see **labs/rooms availability**  
* Teachers book resources or merge sections via the system  
* Admin manages all resources, users, and analytics  
* Notifications sent to students for updates or changes
